# Design source — Clausio

`clausio-pilotage-des-contrats.html` est l'export Claude Design (« Clausio -
Pilotage des contrats ») dont est dérivé le design system du cockpit.

Le langage visuel (palette OKLCH — accent indigo `oklch(0.52 0.13 264)` ≈ `#4b53c9`,
IBM Plex Sans/Mono, rayons 8–14px, ombres teintées indigo, sémantique ok/warn/danger)
est encodé dans `frontend/src/theme/tokens.ts`. Re-skin = modifier ces jetons.
